## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Mahak-agrawal05/coding-ninjas-clone.git
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm run dev
```

Build for production

```bash
npm run build
```
